## 2. 특허 청구 범위 (Patent Specification Draft)

**일상 대화 기반 개인화 영어 학습 시스템 및 쉬운 동사 중심 자연어 변환 방법**

### 핵심 특허 요소 (Claims):

1. **한국어 대화 → 쉬운 동사 영어 변환 파이프라인**
   사용자의 실제 한국어 대화(카카오톡, 음성 녹음 등)를 입력받아, 미국인이 실제 사용하는 쉬운 기본동사(get, put, take, make 등) 및 구동사(phrasal verbs)만을 활용한 자연스러운 영어로 변환하는 AI 파이프라인.

2. **기본동사 + 구동사 쌍 학습 구조 (Verb Pair System)**
   하나의 기본동사와 그로부터 파생된 구동사를 세트로 묶어 학습하는 커리큘럼 구조. 동일 루트 동사의 기본 용법과 확장 용법을 병행 학습하여 활용도를 극대화하는 교육 방법론.

3. **AI 롤플레이 시나리오 자동 생성**
   학습한 동사와 표현을 기반으로 실생활 시나리오(카페 주문, 직장 대화, 친구 약속 등)를 자동 생성하고, 사용자가 해당 상황에서 영어로 대응하는 연습을 제공하는 시스템.

4. **대화 분석 기반 개인화 학습 추천**
   사용자가 입력한 대화 내용을 분석하여 취약 동사(weak verbs)를 식별하고, 해당 동사를 집중 학습하도록 커리큘럼을 자동 조정하는 적응형 학습 엔진.

5. **XP 기반 동기부여 시스템**
   학습 완료, 연속 출석(streak), AI 대화 참여 등 행동에 XP를 부여하고, 레벨 시스템(Beginner → Master)으로 진행도를 시각화하여 지속적 학습을 유도하는 게이미피케이션 구조.
